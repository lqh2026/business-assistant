# 商业知识问答助手

联网搜索 + AI 智能回答的商业问答工具

## 使用方法

### 方法一：双击启动（推荐）
1. 双击 `启动服务.bat`
2. 等待服务启动完成
3. 浏览器自动打开 http://localhost:3000

### 方法二：命令行启动
```bash
# 首次运行需要安装依赖
npm install

# 启动服务
npm start
```

## 配置说明

1. 打开网页后，输入你的 MiniMax API Key
2. API Key 格式：`Group ID:Secret Key`
3. 点击"保存配置"后即可使用

## 功能特点

- 🔍 联网搜索获取实时信息
- 🤖 MiniMax AI 智能回答
- 📚 标注信息来源
- 💼 内置常见商业问题快捷入口

## 注意事项

- 确保 3000 端口未被占用
- 需要联网使用
- API Key 保存在浏览器本地存储中

## 获取 MiniMax API Key

访问 https://www.minimaxi.com/ 注册并获取 API Key
